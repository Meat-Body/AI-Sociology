# v0.2-value-resistance-source

Release title:

AI Sociology v0.2: Value Resistance Source, Weight, and Temporal Closure

## Summary

This release candidate expands the value layer of AI Sociology.

It explains why value should not be reduced to a preference list, and why carbon-based value sovereignty is grounded in embodiment, irreversible time, consequence-bearing, and real choice.

## Core additions

- Value Resistance Source / 价值抗源
- Weight / 重量
- Temporal Closure / 时间关闭性
- Value Constants vs. Value Resistance Source / 价值常量与价值抗源
- The relation between value resistance and sovereignty partitioning
- The relation between value resistance and old-profile lock-in

## Canonical formulas

> 价值不是偏好列表，而是在真实选择中被打出来的。

> 价值抗源是在局部最优诱惑面前保留不可牺牲之物的内核。

> 重量迫使旧排序失效，时间关闭性让选择不可完全回滚。

> 硅基可以模拟代价，但碳基支付时间。
