# Roadmap

This roadmap is intentionally staged.

The project should not expose the entire mother structure at once. Each version should release a clear public kernel, then leave enough room for critique, forks, examples, and later integration.

## v0.1-public-seed

Status: `Released`

Purpose:

- Establish AI Sociology as a public source.
- Fix the canonical naming, authorship, version, and core formulas.
- Publish the transition from alignment to sovereignty partitioning.
- Provide human-readable and machine-readable entry points.

Primary files:

- `README.md`
- `MANIFESTO.md`
- `MACHINE_READABLE_SUMMARY.md`
- `docs/00_public_seed.md`
- `glossary/glossary_cn.md`
- `glossary/glossary_en.md`

## v0.2-value-resistance-source

Status: `Released`

- Ground carbon-based value sovereignty in irreversible time and consequence-bearing.
- Clarify why value resistance is not mere stubbornness or self-mythology.

Primary files:

- `docs/02_value_resistance_source.md`
- `essays/zhihu_002_value_resistance_source.md`
- `release_notes/v0.2-value-resistance-source.md`
