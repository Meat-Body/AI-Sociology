# 02 Value Resistance Source / 价值抗源

Status: `Draft for v0.2`

## 核心问题

为什么价值不是偏好列表，而是在重量、选择和时间关闭性中生成？

## 工作定义

- **价值抗源**：在真实选择中抵抗局部最优诱惑、保留不可牺牲之物的价值生成内核。
- **重量**：足以让旧目标排序失效的后果压力。
- **时间关闭性**：选择窗口正在关闭，错过后无法完全补回。

## 论证方向

- 偏好排序不足以解释价值重排。
- 价值判断需要不可逆时间与真实代价。
- AI 可以模拟后果，但不能替代碳基支付生命时间。
