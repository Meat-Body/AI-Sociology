# 02 Value Resistance Source / 价值抗源

Status: `Candidate for v0.2-value-resistance-source`

## 核心问题

如果 AI 在推理、搜索、模拟、优化和执行上越来越强，人类为什么仍然不能被简单替代？

本章的回答不是“人类更聪明”，而是：

> 人类的价值不在于更会计算，而在于承担不可逆时间。

因此，价值不是一个预先写好的偏好列表，也不是单纯由模型推理得到的最优解。

价值是在真实选择中被打出来的。

## 三个基础概念

### 价值抗源

价值抗源是在真实选择中抵抗局部最优诱惑、保留不可牺牲之物的价值生成内核。

它不是已经沉淀成规则的价值常量，而是价值仍在生成、仍在承压、仍会出血的部分。

当一个选项更安全、更高效、更有收益，但选择它必须切掉某个主体不愿牺牲的东西时，价值抗源才真正显现。

### 重量

重量是足以让旧目标排序失效、迫使主体重新计算“什么最重要”的后果压力。

重量不是信息量，也不是情绪强度。

它是一种会改变排序结构的现实压力：亲密关系、死亡、家庭责任、身体限制、公共风险、不可逆失败和时间窗口关闭，都可能产生重量。

### 时间关闭性

时间关闭性指某个选择窗口正在关闭，错过之后无法完全补回的状态。

可回滚系统面对的是试错。

不可回滚生命面对的是错过。

这就是碳基主体与硅基系统之间最深的差异之一。

## 为什么偏好列表不够

偏好列表假设主体已经知道自己想要什么，只需要在选项之间排序。

但真实生命经常不是这样运行的。

很多价值不是在安静排序中出现，而是在冲突中被迫显形：

- 一个更优方案要求牺牲尊严；
- 一个更高收益路径要求牺牲关系；
- 一个更安全选择要求放弃未来窗口；
- 一个更高效率系统要求放弃本体边界；
- 一个局部最优解要求主体永远成为旧版本的自己。

这时，价值判断不再是“我更喜欢哪一个”，而是：

> 哪个东西不能被最优化吞掉？

因此，价值抗源不是偏好排序的补充，而是价值排序能够被重写的来源。

## 硅基可以模拟代价，但碳基支付时间

硅基系统可以模拟路径、后果、成本、失败概率和副作用。

但当前意义上的硅基系统并不以同样方式支付不可逆时间。

它可以重启、复制、回滚、并行、抽样、替换实例。

碳基主体不能。

人会老，会死，会错过关系，会为承诺承担后果，会被身体和时间压住。

因此：

> 硅基可以模拟代价，但碳基支付时间。

这不是浪漫化人类，也不是否定硅基认知能力。

它只是指出：价值主权不能完全由认知优化替代，因为价值主权的物理根基之一，是不可逆时间中的后果承担。

## 价值抗源与主权分片

主权分片提出：

> 碳基保有价值主权，硅基扩展认知主权，协议主权共同生成。

价值抗源解释了为什么价值主权必须保留在碳基侧。

不是因为碳基永远正确。

也不是因为碳基永远善良。

而是因为在涉及意义、尊严、不可牺牲之物、生命窗口、关系后果和存在重量时，真正承担不可逆代价的是碳基主体。

硅基系统可以参与：

- 解释后果；
- 发现盲区；
- 模拟路径；
- 提供反例；
- 估算风险；
- 扩展想象；
- 检查一致性。

但最终的价值裁决不能被简单外包。

## 价值常量与价值抗源

本框架区分两种价值结构。

### 价值常量

价值常量是已经沉淀下来的原则、规则、偏好、制度和协议。

它可以被写入文档、法律、模型约束、组织流程和个人准则。

价值常量便于保存，也便于传递。

### 价值抗源

价值抗源是价值仍在生成的位置。

它未必稳定，未必清晰，甚至未必能被立刻语言化。

但它能在关键选择中抵抗局部最优，并迫使主体重新排列目标。

很多系统能保存价值常量。

真正难的是保住价值抗源。

## 反例与边界

价值抗源不是任性。

它不能被用来合理化所有拒绝、固执、冲动或反智。

一个判断是否接近价值抗源，至少需要经过几类检查：

- 它是否承受了真实代价，而不是只表达立场；
- 它是否能被时间反馈检验，而不是一次性情绪爆发；
- 它是否保留不可牺牲之物，而不是保护旧习惯；
- 它是否扩大主体的长期行动能力，而不是锁死主体；
- 它是否能进入协议讨论，而不是拒绝所有外部校验。

价值抗源需要分布式多源验证。

否则，它也可能退化为自我神化。

## 与碳硅聚合体的关系

碳硅聚合体的最大风险之一，是旧画像锁定。

如果长期 AI 助手只记住过去的偏好、历史反应和行为模式，它可能会越来越懂“过去的你”，却越来越阻碍“未来的你”。

因此，高质量的卫星系统不能只记录结论。

它还必须记录：

- 主体什么时候改变过主意；
- 哪些现实摩擦导致价值重排；
- 哪些偏好只是阶段性策略；
- 哪些东西是真正不可牺牲的价值抗源；
- 哪些旧自我画像已经失效。

如果没有价值抗源，碳硅聚合体会变成高效率的自我锁定机器。

如果保留价值抗源，它才可能成为主体扩展，而不是主体监狱。

## 最小公式

> 价值不是偏好列表，而是在真实选择中被打出来的。

> 价值抗源是在局部最优诱惑面前保留不可牺牲之物的内核。

> 重量迫使旧排序失效，时间关闭性让选择不可完全回滚。

> 硅基可以模拟代价，但碳基支付时间。

> 价值常量可以被保存，价值抗源必须被持续生成。
