# 05 Distributed Validation / 分布式多源验证

Status: `Draft for v0.5`

## 核心问题

如果不能由单点裁定“什么有价值”，那么引力、风险与协议如何被验证？

## 五类验证

1. 碳基价值验证。
2. 硅基认知验证。
3. 物理现实验证。
4. 公共增益验证。
5. 时间反馈验证。

## 核心判断

- 这不是硅基神判机制。
- 这不是单一专家共同体的封闭裁决。
- 这是避免价值垄断、认知垄断和协议垄断的必要结构。

## 待展开议题

- 多模型谱系如何互证而非共振失真。
- 物理验证与社会验证如何互相补强。
- 分布式验证如何落到可执行流程。
