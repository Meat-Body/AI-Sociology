# Watch Mode / 看守模式

This repository is in watch mode.

The core protocol is technically frozen.

Maintainer activity is limited to:

- fixing broken links;
- correcting typos;
- improving machine readability;
- clarifying glossary conflicts;
- preserving de-identification boundaries;
- maintaining contribution entry points;
- separating core material from appendix material.

The maintainer should not expand the core doctrine due to conceptual excitement.

New ideas should go to `docs/09_protocol_growth_backlog.md`, `rfc/`, local notes, or external forks.

The protocol should be tested by reality, not continuously enlarged by its initiator.
