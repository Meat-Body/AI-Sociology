# RFC-002: Value Resistance Source

Status: `Draft`

## Summary

Explain why value is generated through weight, real choice, and temporal closure rather than reducible preference ordering alone.

## Motivation

Without a stronger account of value formation, "value alignment" remains underspecified and vulnerable to abstraction drift.

## Questions

- What is the minimum condition for a value judgment?
- How do weight and temporal closure change optimization?
- What can silicon simulate but not replace?

## Open Issues

- Edge cases involving reversible choices.
- Relationship to moral philosophy and phenomenology.
- Operational markers for real-world case studies.
